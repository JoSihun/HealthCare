# HealthCare Service
- 헬스장 운영을 위한 웹 서비스


---
# 구현기능 Total 8 Weeks
### 1~2 Week
```
1) 관리자관리
  ㄴ 트레이너 등록
  ㄴ 트레이너 정보 열람
2) 회원관리
  ㄴ 회원 등록(이름, 연락처, 주소)
  ㄴ 회원 정보 열람
```
### 2~3 Week
```
3) 식단추천
  ㄴ 회원별 BMI에 따른 식단추천
4) 루틴추천
  ㄴ 회원별 BMI에 따른 운동추천
```
### 2 Week
```
5) 고객지원 서비스
  ㄴ 라이브챗 시스템(Support System)
```
### 2 Week
```
6) 게시판
  ㄴ 헬스장 소개
  ㄴ 트레이너 소개
  ㄴ 자유게시판
  ㄴ Q&A
  ㄴ FAQ
```
